Apache Annotator (incubating)

Apache Annotator provides annotation enabling code for browsers, servers, and humans.

We are currently seeking developers and community members interested in participating in the future of annotation.

Our intended focus areas are:

  • a collaborative community for creating annotation related code
  • a future for Annotator.js-based projects and plugins
  • enabling Web Annotation in Web browsers, Web Publication readers, and the servers that serve them

Join us!

The dev@annotator.incubator.apache.org mailing list is where most of our conversations currently happen. Simply send an email to dev-subscribe@annotator.incubator.apache.org and be sure to say “hi” once you’re in!

Contributing

We’re still getting things organized and we’d love your help!

Web site code uses Apache Subversion and checkout the site’s contents locally:

$ svn checkout https://svn.apache.org/repos/asf/incubator/annotator/

We are still getting our code repository, Jira issue tracker, and Confluence wiki setup. We’ll update this page once it’s all good and ready.

Thanks!
