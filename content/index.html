Apache Annotator (incubating)

Apache Annotator provides annotation enabling code for browsers, servers, and humans.

We are currently seeking developers and community members interested in participating in the future of annotation.

Our intended focus areas are:

  • a collaborative community for creating annotation related code
  • a future for Annotator.js-based projects and plugins
  • enabling Web Annotation in Web browsers, Web Publication readers, and the servers that serve them

Mailing Lists

The dev@annotator.incubator.apache.org mailing list is where most of our conversations currently happen. Subscribe by emailing dev-subscribe@annotator.incubator.apache.org and be sure to say "hi" once you're in!
